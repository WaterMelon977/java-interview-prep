# Java Interview Preparation

This repository contains comprehensive syllabuses and guides for Java backend developer interviews, focusing on production-grade standards and real-world patterns.

## 🚀 Track 1: Core Java
Deep dive into Java fundamentals and internals required for building scalable backend systems.
- **Fundamentals:** Data types, memory allocation, and Java 14+ switch expressions.
- **OOP & Design:** Inheritance, polymorphism, SOLID principles, and loose coupling.
- **Collections:** Internal workings of HashMap, ArrayList, and Concurrent collections.
- **Concurrency:** Multithreading, synchronization patterns, and Java Memory Model.
- **Java 8+:** Functional programming with Streams, Lambdas, and Optional.

## 🍃 Track 2: Spring Boot & REST APIs
Modern backend development using the Spring ecosystem.
- **Core Spring:** IoC, Dependency Injection, and Bean Lifecycle.
- **REST APIs:** Production-grade API design with Spring MVC.
- **Security:** Authentication and Authorization using Spring Security and JWT.
- **Testing:** Unit and Integration testing for robust services.

---
*Targeted for: Junior to Senior Java Developer roles (MNCs & Startups)*
